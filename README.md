# Physics-Informed Neural Networks (PINNs) 极简教程与实战

本项目包含了几个从零开始实现的物理信息神经网络（PINN）经典示例，完全基于纯 PyTorch 实现，并在 Jupyter Notebook 中分步讲解。如果你想快速上手 PINN 求解偏微分方程 (PDE) 甚至发现物理规律，这些代码可以作为非常好的起步模板。

## 什么是 PINN？(原理解析)

在传统的数值计算框架（如有限元方法 FEM、有限差分法 FDM、谱方法等）中，我们需要在空间和时间上划分细致的网格，并通过数值离散格式来推进求解偏微分方程。但在高维空间问题或存在不规则几何边界时，“维度灾难”和网格生成的代价会变得极其昂贵，且针对某些反问题（Inverse Problem）显得力不从心。

**PINN (Physics-Informed Neural Networks)** 是由 M. Raissi 等人在 2019 年正式提出的一种**深度学习、无网格的偏微分方程求解与反演前沿范式**。

它的核心思想优美且直观：
1. **用神经网络充当试探解（Surrogate Model）**：直接让深度神经网络去拟合 $u(x, t) \approx \text{NN}_{\theta}(x, t)$。它的输入是连续的时空坐标（如坐标 $x$ 和时间 $t$），输出就是该点对应的物理状态量（比如速度、压力、温度或波函数）。
2. **将物理定律转化为 Loss（损失）函数**：我们都知道所有偏微分方程都是基于物理守恒定律（如质量、动量守恒的 Navier-Stokes），本质定义了空间和时间各阶导数之间的数学制衡关系（例如 $\frac{\partial u}{\partial t} + u \frac{\partial u}{\partial x} = \nu \frac{\partial^2 u}{\partial x^2}$）。在 PINN 中，我们并不像传统有监督深度学习那样需要庞大的一对一大量真实数据（Real Data），而是让神经网络输出的数学场**强行满足这套物理方程的导数制衡约束规律**以及**边界/初始条件**，从而促使网络自行推演出符合真实物理世界的运行定律。

## PINN 的训练方式与 Loss 组成

在训练一个 PINN 模型时，完整的推演步骤和损失函数（Loss）由下面几块关键部分拼合而成，而这些也恰恰正是数学偏微分问题适定必备的三要素：

### 1. 物理配点（Collocation Points）与偏微分方程损失 ($Loss_{PDE}$)
* 我们**不需要真实解**的标签！我们只需要在时空计算域 (Domain) 内，随机地或者均匀地撒下大量离散的“测试点”（Collocation Points）。
* 我们将这些测试点塞进神经网络得到预测场，然后依靠深度学习框架内建的 **Automatic Differentiation 自动求导机制**（`torch.autograd.grad`），极其精确地反求出神经网络输出相对于输入坐标 $x$ 和时间 $t$ 的各阶空间和时间导数（完全抛掉了容易受到步长网格影响精度的老旧数值差分手法）。
* 要求这些空间导数，代入原偏微分方程后结果趋近于零！如果方程等式左右不守恒，剩下的残差（Residual）就会作为平方项被加到 **$Loss_{PDE}$** 里面，去“惩罚”网络的权重强行修正，这叫做 **Physics-Informed（受物理先验信息指引约束的）** 回归。

### 2. 初始条件约束 (Initial Condition, IC), $Loss_{IC}$
偏微分模型通常代表随时间演化的物理系统，但它无法凭空预知宇宙最初是什么样的。
* 我们将特定的 $t=0$ 初态分布带入模型，它产生的偏差（相比于人为设定的初始真值），会作为 **$Loss_{IC}$**（初态误差惩罚）。

### 3. 边界条件约束 (Boundary Condition, BC), $Loss_{BC}$
系统不能脱离具体物质和几何约束而存在，热量不会穿透隔绝的物理墙壁。
* 我们在定义好的势阱或墙壁边缘撒下一些有限的点，强行要求网络在此处的温度是 0、波幅是 0、亦或不受热流扰动（视狄利克雷或诺伊曼边界而定）。其方差之和组成了边界惩罚 **$Loss_{BC}$**。

### 总结
总的神经网络训练目标就是三者的调和加权：
$$ Loss_{Total} = w_1 Loss_{PDE} + w_2 Loss_{IC} + w_3 Loss_{BC} $$

网络在没有任何内部真实点监督解析解的情况指引下，只要通过调整数以万计的节点权重（Adam 甚至高阶二阶海森 L-BFGS 下降法）使得这一揽子强硬条件彻底降低收敛到了一定阈值以内，此时得出的**这个网络的非线性激活叠加函数就是该复杂物理方程的一套极佳精确数值解**！

## 仓库实战案例说明

在这个仓库中，不仅实现了理论的基石，更展示了 PINN 的三个强大应用实战模板，你可以通过 Jupyter Notebooks 打开它们分别运行探索（难度从浅入深循序渐进）：

### 1. 简谐运动的常微分方程求解 [`simple_pinn.ipynb`]
最基础地带您第一次了解如何把方程残差拼装成 PDE Loss 放入 PyTorch 优化包中。通过求解一个具有完美解析解的一维函数 $du/dt = \cos(2\pi t)$，最后我们能清晰画出神一样无偏差精确重构解析正弦曲线的纯物理图表。

### 2. 盒子里的量子 - 薛定谔系统求解 [`quantum_pinn.ipynb`]
求解更严肃的物理定态问题：**一维无限深势阱波函数的基态态（量子力学）**。
在这个高能模型中我们发现并解决了一个著名痛点问题——当一整个偏微分方程遇到所有边界和约束**全为 0** 的特殊边界条件时，神经网络容易输出“聪明过头却毫无意义”的**全平庸解（Trivial Solution $u=0$）**。Notebook 内详细注释并探讨了 PINN **如何利用引入先验条件（Peak Loss 孤点强迫）或者积分归一化条件，强势打破 0 解平衡** 并引导 AI 自动定位基态波峰波谷的最优形态，是一次非常具象化的原理解构！

### 3. 流体力学之痛 - Burgers' 方程激波捕捉 [`no_exact_pinn.ipynb`]
这也是早期著名原发论文中的一个核心难点：求解带有极小黏滞系数下的一维黏性 Burgers' Equation（激波对流方程）。**这个强非线性强对流偏微分方程没有闭式的初等解析解**，属于数学界地狱级模拟难题，非常容易在数值模拟产生无穷大爆炸。
为了严谨证明基于梯度的神经网络拥有攻克激波的本领且不是瞎猜出的结果，我们让代码和真正由流体力学高精度差分法耗时演算出来的绝对真集（Ground Truth 原数据位于从 Raissi 实验室下载的 `burgers_shock.mat` 中）同台竞技！
这完美展示验证出了 PINN 极其惊艳的一个巨大优势 —— **在完全不需要去建立复杂高密度精细差分网格（Mesh）的情况下，以稀疏几千个随机数据打点，便能够精确捕获非线性流体的高陡尖锐激波（Shock front）断层。**
